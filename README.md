# Gritty Tech Assessment
This is a technical assessment I completed for my capstone projects sponsor company. 

# How to run
clone this git repo into a desired repository and run these commands
```
cd ai-chat

npm run dev
```
after that you can visit http://localhost:3000 on your desired browser to see the fully functional web app

# How it was made
this entire project was fully made using Next.js and React.js. OpenAI was used for the artificial intelligence portion of the application
* Next.js Documentation - Next.js features and API.
* Learn Next.js - Next.js tutorial.
